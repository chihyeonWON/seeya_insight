학력 : 초대졸 이상(2, 3년 이상)   
나이/성별 : 무관   
경력무관

기술연구소 개발자

보안/헬스케어/기타분야

데이터 수집/정제 및 연계 개발

Linux/Unix 기반 네트워크/서비스 구축 가능자   
정보처리기사, 정보보안기사, 네트워크 관리사, 리눅스마스터 등 관련 자격증 소지자 우대   
Svn, Git 등 버전관리 활용 가능자   
AI 모델 개발 및 튜닝 가능자 (텐서플로 기반)   
능동적인 자세와 활발한 커뮤니케이션
